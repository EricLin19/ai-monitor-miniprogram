## 二、主体类型怎么选（个体户 vs 有限公司）

| 你的目标 | 推荐主体 | 能否做 | 备注 |
|---|---|---|---|
| **纯展示 / 免费信息看板**（不收费、不做交易） | **个体工商户** | ✅ 最低成本，支持绝大多数类目 | 可走"对公打款验证"免 300 认证费（需开对公账户），或直接付 300 元微信认证 |
| **纯展示 + 底部广告（微信流量主）** | **个体工商户** | ✅ 够，无需有限公司 | 广告收入来自广告主、用户免费看内容 → 属"非经营性互联网信息服务"，只需 **ICP 备案**（个体户可办），**不需要 ICP 许可证**，故不需要有限公司。微信官方确认个体户可开通流量主（需累计 UV≥1000、无违规）。类目选"工具"即可 |
| **向用户收费 / 会员 / 付费报告 / 打赏 / 电商**（经营性互联网信息服务） | **有限责任公司** | ✅ 必须，且需 **ICP 许可证**（注册资本 ≥100 万、服务器在大陆、企业法人） | ⚠️ **个体工商户拿不到 ICP 许可证**——只有"向用户直接收钱"才触发此要求；靠广告变现不算 |

> **关键纠正（网传有误）**：不少文章称"个体户认证 30 元/年"，但据微信官方社区，**个体工商户微信认证同样 300 元/年**。个体户"免费"的唯一路径是用**对公账户小额打款验证**（款项原路退回），前提是你开了对公账户；若不开通对公户，就只能付 300。
>
> **微信支付结算户**：个体户可绑定**经营者个人银行卡**（无需对公户）；企业必须绑定对公账户。

> **经营性 vs 非经营性（决定要不要 ICP 证 / 有限公司）**：向**用户**收费 = 经营性 → 需 ICP 许可证 → 必须有限公司；靠**广告**变现、用户免费看 = 非经营性 → 只需 ICP 备案 → 个体户即可。微信流量主官方支持个体户开通，广告组件结算可绑经营者个人卡。

**给小林的建议**：现在先做"免费展示版"验证想法 → 注册**个体工商户**即可，成本最低。靠底部广告（微信流量主）变现**不需要**升有限公司；只有当你要对用户**直接收费**（会员 / 付费报告 / 电商）时，才需升级有限公司并办 ICP 许可证，两者不冲突。

## 三、个体工商户注册实操（线上，约 0.5 天、免费）

> 目标：拿到营业执照 + 统一社会信用代码，用于微信小程序主体认证。纯线上经营可走"网络经营场所"登记，无需实体店。

### 入口（任选其一）
- 全国经营主体登记注册服务网：http://dj.samr.gov.cn → "我要开办"
- 你所在省的政务服务网 / 省企业登记全程电子化平台（如"XX 省企业登记网络服务平台""掌上登记"APP）

### 一步一步操作清单

- [ ] **Step 1 · 进入登记系统**
  - 打开上述入口 → 选"个体工商户设立登记 / 我要开办"
  - 用经营者（你本人）手机号 + 实名账号登录

- [ ] **Step 2 · 名称自主申报**
  - 行政区划：`XX 市 XX 区`
  - 字号：建议 2–4 字（如"智瞰""景析""数衡""观澜"），先系统查重
  - 行业：信息技术 / 信息咨询 / 互联网数据服务
  - 组织形式：`经营部` / `工作室` / `信息技术服务部`
  - 完整示例：`杭州市余杭区智瞰信息技术服务部`
  - 提交预核名 → 保留期通常 30 天

- [ ] **Step 3 · 填写设立登记信息**
  - 经营者：姓名 / 身份证号 / 手机号（本人）
  - 经营场所（二选一）：
    - 纯线上 → 选"**网络经营场所（网址）**"，填你小程序/网店网址 + 平台证明（无需实体店铺）
    - 有实际地址 → 填房产证 / 租赁合同地址
  - 经营范围（勾选，建议含）：
    - ✅ 信息技术咨询服务
    - ✅ 互联网数据服务
    - ✅ 社会经济咨询服务
    - ⚠️ **绝对不要勾**"证券 / 期货 / 投资咨询""金融信息服务"——会触发金融牌照，个体户也拿不到
  - 组成形式：个人经营
  - 资金数额：个体户不验资，填 1–10 万即可

- [ ] **Step 4 · 实名认证 + 电子签名**
  - 经营者用"登记注册身份验证"APP / 政务服务 APP 人脸识别
  - 在申请书电子档上签名（系统自动生成《个体工商户登记(备案)申请书》）

- [ ] **Step 5 · 提交审核 → 领照**
  - 提交 → 市场监管部门审核
  - 多数地区 **0.5–1 个工作日出电子营业执照**，免费
  - 下载电子营业执照（微信"电子营业执照"小程序可亮照），如需纸质可邮寄

- [ ] **Step 6 ·（可选）开立对公账户**
  - 目的：走"对公打款验证"免 300 元微信认证费（款项原路退回）
  - 带营业执照 + 身份证 + 公章（部分地区个体户可免章）到银行对公开户
  - 不开对公户也行 → 小程序认证时直接付 300 元/年

> **保存好**：统一社会信用代码（18 位）、营业执照照片 —— 小程序认证要用。

## 四、小程序主体注册 + 认证 + 备案 + 微信支付

**1. 注册小程序**
- 微信公众平台 https://mp.weixin.qq.com → 立即注册 → 选"小程序" → 填未注册过的邮箱 / 密码
- 主体类型选"**企业**"，企业类型选"**个体工商户**"（或"企业"）
- 填营业执照、经营者身份证；管理员扫码绑定

**2. 主体验证（二选一）**
- **对公账户打款验证**：用对公户向腾讯小额打款（免费，原路退回）→ 适合已开对公户的个体户
- **微信认证**：付 **300 元/年** → 不强制开对公户，最省事

**3. 小程序备案（硬性，发布前必须）**
- 在微信公众平台提交**小程序备案**（2023-09 起强制），需营业执照、负责人身份证、手机号、服务类目、承诺书
- 腾讯初审后转通信管理局，审核约 1–20 个工作日
- **未备案无法上架**

**4. 开通微信支付（如需变现）**
- 个体户：结算户可绑**经营者个人银行卡**
- 企业：必须绑**对公账户**

**5. 类目选择**
- 归到 **"工具"** 或 **"资讯 - 信息资讯服务"**
- ⚠️ 别用"新闻 / 快讯 / 资讯播报"等词 → 会触发《互联网新闻信息服务许可证》（极难拿）
- 用"数据看板 / 景气监测 / 行业数据"等表述
- 全程**只陈述数据、不给买卖建议**，避开"买入 / 卖出 / 超配 / 目标价 / 值得抄底"等投顾措辞

## 五、微信流量主开通实操（纯展示 + 底部广告变现路径）

> 结论前置：个体户 + 工具类目即可开通，不需要 ICP 许可证、不需要有限公司。广告由腾讯审核填充，你只是发布方，合规风险最低。

### 开通前提（必须满足）
- [ ] 小程序已**发布**且完成**备案**（第四条第 3 步）
- [ ] 累计独立访客 **UV ≥ 1000**（流量主硬门槛，得先攒用户）
- [ ] 近 90 天**无违规**记录（无被处罚 / 类目不符等）
- [ ] 类目合规（建议"工具"，已照第四条第 5 步选好）
- [ ] 已绑定结算账户（个体户绑**经营者个人银行卡**，见第四条第 4 步）

### 一步一步操作清单

- [ ] **Step 1 · 申请入口**
  - 微信公众平台 → 左侧菜单 **"流量主"** → 点击"开通流量主"
  - 同意《微信小程序广告组件运营规范》《流量主服务协议》

- [ ] **Step 2 · 选择广告位**
  - 推荐先开 **"底部 Banner 广告"**（贴合你说的"最下面放个广告"）
  - 后续可加：激励视频、插屏、视频贴片、原生模板（按需）
  - 系统生成 **广告位 ID（ad unit-id）**，复制保存

- [ ] **Step 3 · 绑定结算**
  - 个体户：结算户选"**经营者个人银行卡**"（无需对公户）
  - 填写开户行、卡号、纳税人信息（个体户按小规模纳税人）
  - 企业主体才需寄增值税专用发票给腾讯；个体户走自然人结算更省事

- [ ] **Step 4 · 腾讯审核**
  - 广告组件 / 账户资质审核通常 **1–3 个工作日**
  - 审核通过 → 广告位状态变为"已启用"

- [ ] **Step 5 · 小程序代码接入广告**
  - 在目标页面（如首页 `index.wxml` 最底部）加组件：
    ```xml
    <!-- 底部 Banner 广告 -->
    <ad unit-id="你的广告位ID" ad-intervals="30" ad-type="banner"></ad>
    ```
  - 注意：`ad` 组件**只能放在页面非 scroll-view 区域的固定位置**，真机预览确认正常展示
  - 提交审核 → 重新发布版本

- [ ] **Step 6 · 数据看板 & 收款**
  - 流量主后台看"曝光 / 点击 / 收入 / eCPM"实时数据
  - 腾讯**按月结算**（次月 1–15 日核对，15 日后打款到绑定卡）
  - 税务：广告收入属经营所得，按小规模纳税人报（当前季度 30 万内免增值税，以 2026 年当期政策为准）

### ⚠️ 流量主合规红线（务必避开）
- **别自己接第三方"广告主"**：尤其**别直接承接券商 / 基金 / 理财类金融广告**——属金融产品推广，要审查广告主资质，极易踩金融广告合规红线。老老实实走**微信官方流量主**，广告由腾讯审核填充最稳。
- **内容仍保持客观**：你的景气数据若被判定为"金融服务"类目，流量主也可能要求额外资质。所以第三 / 四条里的文案红线照改，类目锁死在"工具"。
- 广告组件不得遮挡功能、不得诱导点击（"点这里领红包"之类违规）。
